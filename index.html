A bus is a group of wires that connects all parts of a computer and carries data between them. The single bus connects the ALU and ALL registers via one common internal path.

Q9 Write short note on Stack Memory and its Operations

What is a Stack? ‚Üó GeeksforGeeks

A stack is a special memory structure that follows the LIFO (Last In, First Out) principle. Think of a pile of plates ‚ÄĒ you always add and remove from the TOP!

Key Terms

  • TOP ‚ÄĒ where items are added/removed
  • BOTTOM ‚ÄĒ the very first item placed
  • SP (Stack Pointer) ‚ÄĒ register that always points to current TOP

PUSH Operation (Adding to stack)

Subtract SP, SP, #4 ‚Üź SP moves DOWN (stack grows downward) Store Rj, (SP) ‚Üź new item placed at TOP

POP Operation (Removing from stack)

Load Rj, (SP) ‚Üź take TOP item into register Rj Add SP, SP, #4 ‚Üź SP moves UP (new TOP revealed)

Why Multi-Bus?

Single bus is slow (only 1 transfer per clock). Multi-bus provides multiple internal paths enabling several transfers in parallel ‚Üí significantly faster!

Three-Bus Structure

Bus A ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚Ėļ Bus B ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚Ėļ } to ALU inputs Bus C ‚óĄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ‚ĒÄ ALU output Register file has 3 PORTS: Port A ‚Üí output to Bus A (source operand 1) Port B ‚Üí output to Bus B (source operand 2) Port C ‚Üí input from Bus C (destination result)

How it Works

  • Bus A and B carry TWO source operands to ALU simultaneously
  • ALU performs operation and sends result over Bus C
  • All three happen in the SAME clock cycle!
  • No need for temporary registers Y and Z (unlike single bus)
